## Description

[Merit Tax Advisor](https://dev.ma.tax) node js api.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development & starting the server
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
